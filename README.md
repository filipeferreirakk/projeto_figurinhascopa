# Sistema de Album de Figurinhas - Copa 2026

Projeto da disciplina de Estrutura de Dados. Sistema de console para
colecionar, organizar e trocar figurinhas da Copa do Mundo.

Toda a lista encadeada e as filas foram implementadas do zero usando nos
encadeados, sem usar list, deque ou qualquer estrutura pronta do Python.

## Arquivos

- figurinhas.py - todas as classes do projeto: Figurinha, NodoLista,
  NodoFila, Fila, Album e Historico, alem da validacao de selecoes.
- menu.py - menu do programa que usa as classes.

## Funcionalidades

- Inserir, remover e consultar figurinha
- Ver album completo e porcentagem concluida
- Guardar, mostrar e contar figurinhas repetidas
- Buscar por numero, por jogador e por selecao
- Registrar e efetuar troca de figurinhas repetidas entre dois albuns
- Salvar e carregar os dados em arquivo CSV
- Tratamento de entradas invalidas (numeros e selecoes)

## Como executar

    python menu.py

Depois e so seguir as opcoes do menu pelo teclado.
